WZ Biên Bản

Biên bản họp tiếng Việt, chạy ngay trên máy bạn. Ghi âm 1 lệnh, kết thúc là có biên bản + PDF.

Cần có trước: máy Mac (chip Apple M1 trở lên) · đã cài Claude Code · ffmpeg (mở Terminal gõ brew install ffmpeg) · Google Chrome.

1 Nạp plugin (chạy 1 lệnh)

Mở Terminal và dán đúng 1 dòng này:

claude plugin marketplace add workzone-vn/wz-bien-ban && claude plugin install wz-bien-ban@work-zone

Báo Successfully installed là xong bước này.

2 Mở lại Claude Code + cài 1 lần

Đóng và mở lại Claude Code. Sau đó gõ lệnh sau (chỉ chạy 1 lần đầu, để tải bộ nhận giọng nói):

/cai-dat
Lần này tải khoảng 3GB nên mất vài phút. Lần ghi âm đầu tiên, máy Mac sẽ hỏi quyền Micro cho Terminal, bấm cho phép.

3 Dùng hàng ngày (chỉ 2 lệnh)

/bat-dau-hopBắt đầu ghi âm cuộc họp. (Chưa cài BlackHole thì bật loa ngoài, không đeo tai nghe.)
/ket-thuc-hopDừng ghi → tự transcript → viết biên bản → xuất PDF và mở ra cho bạn.

File kết quả nằm ở thư mục ~/wz-bien-ban/output/<tên cuộc họp>/: bien-ban.pdf (gửi đi), viewer.html (xem có tìm kiếm).

Riêng tư: toàn bộ âm thanh và bản ghi nằm trên máy bạn. Phần viết biên bản dùng chính Claude Code của bạn. Không có file âm thanh nào gửi lên máy chủ bên thứ ba.
